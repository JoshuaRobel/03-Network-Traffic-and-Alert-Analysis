# 03 — Network Traffic & Alert Analysis
## Enterprise IDS Monitoring (Snort + Wireshark)

This repository demonstrates enterprise network monitoring and alert validation within an internal corporate SOC environment.

Alerts are generated by Snort IDS and validated through packet-level inspection using Wireshark.

Each investigation demonstrates:

- IDS alert interpretation
- Packet capture validation
- IOC extraction
- False positive analysis
- MITRE ATT&CK mapping
- Containment decision-making

---

# 🏢 Simulated Enterprise Environment

- Corporate perimeter firewall
- Snort IDS deployed at network boundary
- Internal Windows endpoints
- Active Directory environment
- Segmented internal VLANs
