## Detection Improvements

| Recommendation | Rationale |
|----------------|-----------|
| Add Suricata rule for `.exe` downloads from <3 day old domains | Prevent similar attacks |
| Alert on HR workstation outbound connections to rare TLDs | Higher sensitivity for sensitive roles |
| Email gateway: flag emails with recently registered domains | Catch phishing at entry |

## Lessons Learned

- DNS-aged domain check would have caught this pre-download
- User clicked phishing link within 5 minutes — security awareness gap
- EDR isolation worked as intended, preventing execution
